# unibanner

## About


Unibanner is a small-ish IRC bot built to `ZLINE`{:.highlight} and blacklist

unicode spamming bots like those originating from the freenodegate-type bots


```shell
$ bundle exec unibanner
$ unibanner
# ...
# Unibanner runs
```
